## 1. 전체 흐름 요약

```txt
홈 (home/)
  ↓ 새 사건 + → 카테고리 선택
AI 대화방 (rooms/[roomId])
  ↓ 진술저장 → 모달: 혼자서 진행 / 상대방 초대
  ├── 혼자서 진행 → disputes/[id] (단독 판결)
  └── 상대방 초대 → 초대 링크 발급 → join/ → disputes/[id]/statement (role_b)
사건작성 (disputes/[id]/statement)
  ↓ 진술저장
사건조회 (disputes/[id])
  ├── 진술 탭: 양측 진술 확인 (카드 클릭 시 본인 진술 수정 가능)
  ├── 판결 탭: 판결 결과 (1인: 텍스트만 / 2인: 프로필+그래프+텍스트)
  └── 유형 탭: 결과 유형 캐릭터 + 공유/다운로드
```

### 사건 (Dispute)

| 경로 | 화면명 | 설명 | 인증 |
|------|--------|------|------|
| `/disputes/[disputeId]` | 사건조회 | 진술 탭 / 판결 탭 / 유형 탭 | 🔒 |
| `/disputes/[disputeId]/statement` | 사건작성 | role_a / role_b 진술 작성 및 수정 | 🔒 |

#### 사건조회 탭 상세

| 탭 | 내용 |
|----|------|
| 진술 탭 | 양측 진술 카드 표시. 본인 진술 카드 클릭 시 수정 가능 |
| 판결 탭 (혼자서 진행) | 텍스트 판결결과만 제공. 프로필 사진 / 그래프 없음 |
| 판결 탭 (상대방 초대) | 프로필 사진 + 그래프 + 텍스트 판결결과 제공 |
| 유형 탭 | AI 도출 결과 유형 캐릭터 이미지 및 설명. 공유하기 / 결과 다운받기 버튼 |
